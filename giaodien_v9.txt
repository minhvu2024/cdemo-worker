NOTES:
═══════════════════════════════════════════════════════════════════════════

1. Navigation Bar: Fixed at top, contains 4 main tabs + Gen Data link
2. Dashboard: Shows statistics cards and charts for overview (no progress bars)
3. BIN Checker: Search and filter BINs with pagination, Copy BINs List button
4. Card Exporter: Filter BINs and export cards with preview
5. Tools: 3-column layout with Normalize, Check Duplicates, and Import
6. All tabs use glass morphism design with light theme
7. Progress bars shown during processing operations
8. Real-time card counters in Tools tab textareas
9. File upload support for Check Duplicates and Import
10. Horizontal scroll always at start for textareas
